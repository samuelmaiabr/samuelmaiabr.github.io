Anexo: ‘Complexidade Econômica e Desigualdade de Renda:’

Author

Samuel Maia, Cedeplar, UFMG

Neste documento, você encontrará um registro completo do ambiente computacional utilizado na execução do script wid-SITC-cleaning.R (disponível para baixar aqui. Esse script foi empregado na exploração, limpeza e estruturação dos dados do artigo “Complexidade Econômica e Desigualdade de Renda: a vista do topo é diferente” (Projeto). Para acessar ao relatório do qual este documento é apenas um acréscimo, clique aqui.

Aqui, o objetivo principal é permitir que seja possível:

  • recriar o ambiente de trabalho usado na análise do Projeto
  • identificar versões específicas de pacotes utilizados
  • entender as dependências do sistema
  • validar a integridade do ambiente através do renv

Componentes principais incluídos:

  1. Especificações técnicas do sistema
  2. Inventário completo de pacotes instalados
  3. Mapa de dependências entre pacotes
  4. Estado de conservação do ambiente virtual
  5. Metadados críticos para replicação
